Transcrição

Resolveremos a questão de como o momento_cadastro está sendo apresentado para os usuários nesta etapa.

Sabemos que cada país possui suas representações próprias de datas. Nos Estados Unidos é exibido mês, dia e ano, enquanto no Japão é ano, mês e dia por exemplo. Nos padrões brasileiros, exibiremos o dia, seguido do mês com o ano ao final, como é conhecido com "dd/mm/aaaa".

Em sua biblioteca datetime e a classe datetime, o Python possui o método strftime() que receberá a data e a formatação regida por outros caracteres especiais. É bastante parecido com o que fizemos usando as expressões regulares ou regex.

  • %A retorna os dias da semana por extenso, como Sunday;

  • %d exibe os dias do mês com números de 01 a 31;

  • %m retorna meses em números de 01 a 12;

  • %y mostra o ano com apenas dois dígitos;

  • %Y apresenta o ano em formato de quatro números;

  • %H retorna o horário no formato decimal, como 00, 001 e etc;

  • %M exibe os minutos de forma decimal;

  • %S apresenta os segundos em decimal.

Usaremos estes caracteres especiais para compôr nossa formatação e exibi-la aos usuários.

É interessante que mostremos o dia, o mês e o ano seguido das horas e minutos apenas do momento de cadastro.

Faremos o primeiro teste no main.py comentando o anterior. Criaremos uma variável hoje que será igual a datetime.today(), e outra hoje_formatada sendo hoje.strftime() recebendo a formatação entre aspas.

Começaremos com uma simples formatação do ano com "%Y" para vermos o resultado. Por fim, printaremos ambos.

from datetime import datetime, timedelta
from datas_br import DatasBr

#cadastro = DatasBr()
#print(cadastro.momento_cadastro)

hoje = datetime.today()
hoje_formatada = hoje.strftime("%Y")

print(hoje)
print(hoje_formatada)

Rodando o código, teremos o objeto datetime e uma representação somente com o ano.

Na sequência, adicionaremos %d e %m antes do ano separados por barras para fazer a formatação.

from datetime import datetime, timedelta
from datas_br import DatasBr

#cadastro = DatasBr()
#print(cadastro.momento_cadastro)

hoje = datetime.today()
hoje_formatada = hoje.strftime("%d/%m/%Y")

print(hoje)
print(hoje_formatada)

O resultado no console já estará mais parecido com a forma que estamos acostumados a ver no Brasil.

Adicionaremos também %H:%M ao strftime() para obtermos o horário no formato correto. Desta forma, teremos a data e o horário como queremos.

É importante sabermos qual é o tipo da variável hoje_formatado, imprimindo seu type().

from datetime import datetime, timedelta
from datas_br import DatasBr

#cadastro = DatasBr()
#print(cadastro.momento_cadastro)

hoje = datetime.today()
hoje_formatada = hoje.strftime("%d/%m/%Y %H:%M")

print(hoje)
print(hoje_formatada)
print(type(hoje_formatada))

Com isso, veremos que se trata de uma str; ou seja, dessa forma a variável perde todos os métodos que estávamos usando como month(), weekday() e outros da biblioteca.

Portanto, é interessante usarmos essa metodologia somente para o final visto pelas usuárias e usuários. Afinal, não trabalharemos mais com a data após formatá-la.

Indo ao datas_br.py , criaremos um novo método format_data que receberá somente o self. Em seguida, a data_formatada será igual ao momento_cadastro e poderemos aproveitar o código de strftime().

Com isso, pegaremos a data cadastrada para formatá-la. Por fim, a imprimiremos para podermos retornar ao main.py.

from datetime import datetime

class DatasBr:
    def __init__(self):
        self.momento_cadastro = datetime.today()

    def mes_cadastro(self):
    meses_do_ano = [
        "janeiro", "fevereiro", "março",
        "abril", "maio", "junho",
        "julho", "agosto", "setembro",
        "outubro", "novembro", "dezembro"
    ]

        mes_cadastro = self.momento_cadastro.month -1
        return meses_do_ano[mes_cadastro]

    def dia_semana(self):
    dia_semana_lista = [
        "segunda", "terça",
        "quarta", "quinta", "sexta"
    ]

        dia_semana = self.momento_cadastro.weekday()
        return dia_semana_lista[dia_semana]

    def format_data(self):
        data_formatada = self.momento_cadastro.strftime("%d/%m/%Y %H:%M")

Ter o return de data_formatada é interessante para podermos jogar format_data() para dentro de um método __str__(), sem precisarmos chamar outro método sempre que quisermos representar nossa formatação correta.

O retorno será self.format_data() e não precisaremos passar nada, pois estamos trabalhando somente com nosso atributo.

from datetime import datetime

class DatasBr:
    def __init__(self):
        self.momento_cadastro = datetime.today()

        def __str__(self):
            return self.format_data()

//código omitido

    def format_data(self):
        data_formatada = self.momento_cadastro.strftime("%d/%m/%Y %H:%M")
        return data_formatada

De volta ao main.py, poderemos simplesmente printar a instância da nossa classe.

O retorno será de acordo com o que esperávamos, pois criamos a máscara na formatação, o inserimos em __str__() e temos a representação str de nosso objeto.

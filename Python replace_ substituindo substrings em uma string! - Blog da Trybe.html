Quando falamos em Python, o Python replace() pertence ao conjunto de mtodos disponveis na linguagem e utilizado para a manipulao de variveis do tipo string. Sua utilizao essencial para trocar uma ou mais trechos de seu contedo e o resultado disponibilizado em uma nova string.

Python replace: o que e para que serve esse mtodo?

O mtodo replace() utilizado para substituir um ou mais trechos em uma string. Ele contm parmetros para auxiliar a forma de substituio desse contedo. Em Python, o tipo string imutvel, o que significa que quando ela declarada, no possvel realizar alteraes em seu contedo.

Em funo disso, sempre que um mtodo de manipulao de string executado, entre eles a funo replace(), o resultado gerado em uma nova varivel que, nesse caso, tambm do tipo string.

Qual a sintaxe desse mtodo?

O mtodo replace() contm dois parmetros obrigatrios e um opcional, falaremos sobre cada um mais adiante. Sua sintaxe :

string.replace(oldvalue, newvalue, count)

Conhea os parmetros do mtodo replace

Como mencionamos, o mtodo replace() utilizado para substituir uma ou mais ocorrncias de um determinado contedo em uma string. Para isso, precisamos informar quais so esses valores por meio de parmetros da funo. Confira como funciona cada um deles, a seguir.

Oldvalue

O parmetro oldvalue corresponde ao valor original que ser substitudo na string. Na prtica, o compilador percorrer toda a string e, quando encontrar a substring correspondente, vai substitu-la pelo contedo informado em newvalue. Portanto, esse um parmetro obrigatrio. Veja um exemplo:

frase = "Linguagens de programao: Python, C, Java, JavaScript."
print(frase.replace(",", " -"))

#Resultado: Linguagens de programao: Python - C - Java - JavaScript.

No exemplo acima, toda a ocorrncia do caractere vrgula na string frase ser substitudo pelos caracteres de espao e hfen.

Newvalue

O parmetro newvalue trabalha em conjunto com o oldvalue. Na prtica, ele que informa ao compilador qual ser o novo contedo inserido no trecho da string definido no parmetro oldvalue. Perceba que, no exemplo anterior, todas as vrgulas foram substitudas pelo novo valor: .

Count

O parmetro count opcional e indica quantas vezes ser feita a substituio da substring definida. Veja um cdigo de exemplo:

frase = "Linguagens de programao: Python, C, Ruby, JavaScript."
print(frase.replace(",", " -",2))

#Resultado: Linguagens de programao: Python - C - Ruby, JavaScript.

Perceba que definimos o valor de count igual a 2. Assim, a vrgula foi substituda pela substring indicada apenas nas duas primeiras ocorrncias e se manteve entre as linguagens Ruby e JavaScript. importante dizer que, se o parmetro count for omitido ao chamar a funo replace(), todas as ocorrncias da substring sero substitudas pelo valor correspondente.

Exemplos de uso do mtodo replace

Agora que j vimos a parte terica do mtodo replace(), vamos desenvolver alguns exemplos prticos para demonstrar como esse recurso funciona.

Remover quebras de linhas em uma string

Uma forma de utilizao do mtodo replace() para a remoo dos caracteres correspondentes quebra de linha \n. Nesse caso, podemos substituir essa ocorrncia por um caractere de espao. Veja um exemplo:

linguagens="Python\nJavaScript\nC\nRuby"
print(linguagens.replace("\n", ", "))

#Resultado: Python, JavaScript, C, Ruby

No exemplo acima, substitumos a ocorrncia da quebra de linha pelos caracteres de vrgula e espao.

Alterar um determinado trecho da string

Podemos utilizar a funo replace() para modificar apenas uma determinada palavra em uma string. No exemplo abaixo, trocamos a linguagem C por Rust. Veja o algoritmo:

frase= "Linguagens de programao: Python, C, Ruby, JavaScript."
print(frase.replace("C", "Rust"))

#Resultado: Linguagens de programao: Python, Rust, Ruby, JavaScript.

Alterar vrias ocorrncias de uma substring

Como mencionamos, podemos definir o nmero de vezes que determinada substring ser alterada. Para isso, utilizamos o parmetro count no mtodo replace(). Veja um exemplo:

numeros = "um dois trs quatro cinco"
print(numeros.replace(" ", "\n", 3))

'''
Resultado:
um
dois
trs
quatro cinco
'''

No cdigo fonte acima, trocamos o caractere de espao pelo de nova linha apenas em 3 ocorrncias. Por isso, o contedo quatro cinco foi exibido em uma mesma linha.

Como substituir trechos de uma string utilizando expresses regulares?

O mtodo replace() s pode ser utilizado com um parmetro de seleo. Isso significa que se quisermos, por exemplo, localizar em uma string a ocorrncia do caractere dois pontos e tambm da vrgula, no ser possvel com esse recurso. Entretanto, podemos utilizar expresses regulares para nos ajudar nessa tarefa.

Vale ressaltar que as expresses regulares correspondem a um recurso muito utilizado em diversas linguagens de programao. No entraremos em detalhes sobre sua elaborao, pois temos este guia inicial com dicas valiosas sobre o assunto, que voc pode conferir mais tarde.

Para utilizar expresses regulares Python regex replace  para substituir partes de uma string, precisamos utilizar o mdulo re, que uma abreviatura de regular expression. Na prtica, esse mdulo contm uma srie de mtodos que possibilitam a utilizao de expresses regulares com diferentes tipos de dados, entre eles, a string.

Portanto, para substituir um determinado trecho de uma string devemos usar o mtodo re.sub() que tem a seguinte sintaxe:

re.sub(pattern, replacement, string, count, flags)

Em que:

  • pattern: representa uma expresso regular capaz de identificar determinada substring na varivel de origem;
  • replacement: representa a substring que ser substituda ou uma funo que retorne uma string;
  • string: indica a string original;
  • count: indica o nmero de repeties, ou seja, de possveis substituies caso exista o valor correspondente na string. Esse parmetro opcional;
  • flags: tambm um parmetro opcional e representa os sinalizadores de expresses regulares. Por padro, nenhum valor aplicado. Entretanto, podemos utilizar, por exemplo, a flag = re.I, que equivale instruo re.IGNORECASE. Assim, no ser feita distino entre letras maisculas ou minsculas.

Nada melhor que conferir alguns exemplos prticos para visualizarmos como esse recurso funciona e sua diferena em relao ao mtodo replace(). Vamos l!

Alterar partes do endereo de e-mail

Podemos utilizar o mtodo re.sub() com uma expresso regular para alterar uma parte do endereo de e-mail. Veja o exemplo abaixo:

import re

email = 'joaopedro@dominioa.com mariaemilia@dominiob.d1 paulosergio@dominio3.com suelisilva@dominiod'
resultado = re.sub(r'@(\w*)(.\w{2,3})', '@novodominio.com.br', email)
print(resultado)

'''
Resultado:
joaopedro@novodominio.com.br mariaemilia@novodominio.com.br.br paulosergio@novodominio.com.br suelisilva@novodominio.com.br
'''

Perceba que logo na primeira linha de cdigo realizamos a importao do mdulo re, pois ele essencial para que o mtodo re.sub() funcione. A seguir, temos a varivel email que contm alguns endereos separados por espaos e com domnios diferentes.

Na sequncia, criamos a varivel resultado para armazenar o retorno da funo re.sub(). Veja que utilizamos uma expresso regular para selecionar o trecho do e-mail aps o caractere @. Dessa forma, podemos substituir esse trecho pelo novo valor, que corresponde a @novodominio.com.br.

Por fim, exibimos o resultado na tela com todos os endereos modificados para o domnio definido na funo re.sub(). Vale ressaltar que, antes de escrevermos a expresso regular, utilizamos o caractere r que representa uma string bruta.

Isso necessrio para garantir que a expresso ser lida da forma em que foi escrita, pois a linguagem Python interpreta a barra invertida \ e o caractere seguinte como especiais, podendo ser uma nova linha, tabulao etc. Isso pode gerar alguma confuso na interpretao da instruo.

Alterar substring a partir de duas alternativas

Se tivermos uma string em que desejamos utilizar duas ou mais substrings para a seleo do trecho a ser alterado, podemos aplicar o mtodo re.sub() para facilitar essa tarefa. Veja o exemplo abaixo:

import re

linguagem= "Linguagens de programao: Python, C, Ruby, JavaScript."
print(re.sub(r':|,', ' -', linguagem))

#Resultado: Linguagens de programao - Python - C - Ruby - JavaScript.

No exemplo acima queremos substituir dois caracteres: o dois pontos e a vrgula pelos caracteres de espao e hfen. Perceba que no mtodo re.sub() utilizamos a expresso :|,, ou seja, usamos o operador | (ou) para dizer ao compilador que ele deve substituir a substring ao encontrar uma das duas alternativas.

Retornar o resultado em uma tupla

Em determinadas situaes, pode ser necessrio ter o conhecimento de quantas vezes a substituio foi feita em uma string. Isso possvel por meio do mtodo re.subn(), que equivalente funo re.sub(), mas retorna uma tupla com dois elementos. O primeiro corresponde string modificada e o segundo, ao nmero de vezes em que a alterao foi aplicada. Veja um pequeno exemplo a seguir:

import re
 
email = 'joaopedro@dominioa.com mariaemilia@dominiob.d1 paulosergio@dominio3.com suelisilva@dominiod'
lista_email = re.subn(r'@(\w*)(.\w{2,3})', '@novodominio.com.br', email)
print(lista_email)

'''
Resultado:
('joaopedro@novodominio.com.br mariaemilia@novodominio.com.br paulosergio@novodominio.com.br suelisilva@novodominio.com.br', 4)
'''

for pessoa in lista_email:
   print(pessoa)

'''
Resultado:
joaopedro@novodominio.com.br mariaemilia@novodominio.com.br paulosergio@novodominio.com.br suelisilva@novodominio.com.br
4
'''

No algoritmo acima, utilizamos a mesma lgica de programao que consta no exemplo em que alteramos o domnio do e-mail. A diferena que aplicamos o mtodo re.subn() para fazer as substituies necessrias.

Perceba que, ao depurarmos o cdigo, tivemos como retorno uma tupla chamada lista_email com dois elementos, no qual o primeiro corresponde string com os endereos de e-mail alterados para o novo domnio, e o segundo, equivale ao nmero de vezes que a substituio foi realizada.

Por fim, utilizamos uma estrutura de repetio for para percorrer os elementos da tupla lista_email e exibi-los na tela.

O mtodo Python replace usado para substituir trechos de uma string por uma ou mais vezes. Tambm podemos utilizar o mdulo re, no qual podemos aplicar expresses regulares para refinar ainda mais as condies de substituio. Portanto, a linguagem oferece diversos recursos para a manipulao de strings.

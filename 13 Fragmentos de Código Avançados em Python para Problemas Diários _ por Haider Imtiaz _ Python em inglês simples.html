1. Marca d'água em uma imagem

Este trecho de código ajudará você a marcar qualquer imagem com marca d'água. Para isso, você precisa instalar o OpenCV.

# Instale esta biblioteca "pip install opencv-python"import cv2 
 
original_img = cv2.imread('img_500x326.jpg', 1)water_mark = cv2.imread('watermark_500x326.jpg', 1) 
 
img = cv2.addWeighted(water_mark, 0.1, original_img, 1.0, 0) 
 
cv2.imwrite('Watermark.png',img) 
 
cv2.waitKey(0) 
 
cv2. destroyAllWindows()

4. Criptografar/Descriptografar String

Você pode criptografar e descriptografar String com o seguinte trecho de código Python.

# Criptografia e Descriptografia de Stringfrom cryptography.fernet import Fernet 
 
key = Fernet.generate_key() 
cipher_suite = Fernet(key) 
 
# Código codificado 
encoded_text = cipher_suite.encrypt(b"Bem-vindo de volta Desenvolvedor Python") 
print("encoded text: ", encoded_text) 
 
# Código Decodificado 
decoded_text = cipher_suite.decrypt(encoded_text) 
print("decoded text: ", decoded_text)# Saída: 
# texto codificado: b'gAAAAABhdbmXBR1_EuQ_HzcomFT.......BUX1HZjAqaE='

10. Resize an Image

You don’t need any web app or software now you can resize any image with the following snippet code.

import cv2
 
img = cv2.imread('Image.jpg', cv2.IMREAD_UNCHANGED)scale_percent = 50 # change accordingly
 
width = int(img.shape[1] * scale_percent / 100)
height = int(img.shape[0] * scale_percent / 100)
 
dimension = (width, height)resized = cv2.resize(img, dimension, interpolation = cv2.INTER_AREA)cv2.imwrite('resized.jpg', resized) # Image savedcv2.waitKey(0)cv2.destroyAllWindows()
